
# Account Name

The first and last name of an account owner

## Structure

`AccountName`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `First` | `string` | Required | The first name of the account owner |
| `Last` | `string` | Required | The last name of the account owner |

## Example (as JSON)

```json
{

  "last": "last0"
```
