
# V1 Pipelines Keys Request

## Structure

`V1PipelinesKeysRequest`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Name` | `string` | Optional | A name for the Trigger Key. |
| `Ips` | `List<string>` | Optional | An array of ips this Trigger Key is usable from. |

## Example (as JSON)

```json
{
  "name": "name4",
  "ips": [
    "ips8"
  ]
}
```
