
# Name

The first and last name associated with the Account.

## Structure

`Name`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `First` | `string` | Required | The first name of the Account owner. |
| `Last` | `string` | Required | The last name of the Account owner. |

## Example (as JSON)

```json
{
  "first": "first6",
  "last": "last0"
}
```
