
# Storage

Storage information for the volume.

## Structure

`Storage`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Used` | `int` | Required | The amount of volume storage used, in MB's |
| `Total` | `int` | Required | The total amount of volume storage available, in MB's. |

## Example (as JSON)

```json
{
  "used": 76,
  "total": 242
}
```
