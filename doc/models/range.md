
# Range

A start and end date-time pair indicating a range of time

## Structure

`Range`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Start` | `DateTime` | Required | - |
| `End` | `DateTime` | Required | - |

## Example (as JSON)

```json
{
  "start": "01/30/2021 08:30:00",
  "end": "01/30/2021 08:30:00"
}
```
