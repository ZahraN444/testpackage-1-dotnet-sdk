
# Tls 2

TLS properties of the record.

## Structure

`Tls2`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Enable` | `bool` | Required | A boolean, where true represents this record will be paired with a TLS certificate automatically maintained by the platform. |
