
# Associated Credit

Credit associated with an invoice.

## Structure

`AssociatedCredit`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Id` | `string` | Required | A 24 character hex string used to identify a unique resource. |
| `Amount` | `int` | Required | The amount of the credit. |

{
