
# Order Events

A collection of timestamps for each event in the orders lifetime.

## Structure

`OrderEvents`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Expires` | `DateTime` | Required | The timestamp of when the order expires. |

## Example (as JSON)

```json
{
  "expires": "01/30/2021 08:30:00"
}
```
